# PINN-PyTorch : Notebooks pédagogiques PINN (work in progress)

## Contexte
Formation personnelle et documentation pédagogique.
Implémentation pas à pas de PINN en PyTorch, appliqués aux mêmes
équations que TreeGrass (dynamiques ligneux/herbacés en savane), mais
en 1D pour la clarté pédagogique.

## Contenu
- Notebook 1 : calcul de dérivées partielles avec PyTorch (autograd)
  — exemples simples, progression pas à pas
- Notebook 2 : implémentation PINN 1D sur un système d'EDP
  représentant les dynamiques de biomasse en savane

## Stack
Python - PyTorch - Jupyter - Markdown
