# pydata2pde : Découverte d'EDP à partir de données (PINN + workflow CEDI)

## Contexte
Développement autonome au CIRAD dans le cadre du projet Deep2PDE.
Implémentation d'un workflow complet pour extraire une EDP à partir
de données représentant des dynamiques temporelles.

## Workflow CEDI
- **C**ollection : données synthétiques de biomasse arborée en 3D
- **E**xtrapolation : interpolation spatiotemporelle
- **D**ifferentiation : calcul des dérivées via PINN
- **I**dentification : estimation des paramètres de l'EDP

## Méthodes
- Physics-Informed Neural Networks (PINN) en 3D
- Refonte complète du code (migration TensorFlow, extension 2D→3D)
- Application à la dynamique de biomasse d'arbres

## Stack
Python - TensorFlow - GitLab institutionnel (public)
